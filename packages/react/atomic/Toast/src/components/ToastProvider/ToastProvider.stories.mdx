# Basic usage

```jsx
<ToastProvider>
  const {addToast} = useToast();
  <button
    onClick={() => {
      addToast({
        type: "primary",
        text: "Toast",
        duration: 4000,
      });
    }}
  >
    Toast
  </button>
</ToastProvider>
```

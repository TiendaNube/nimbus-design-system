# Changelog

The TimePicker component allows users to select a time value using either a scroll-based wheel picker or a dropdown list of time options.

## 2026-01-16

### Added

- Initial release of TimePicker component
- Support for `scroll` and `dropdown` picker modes
- Support for `12h` and `24h` time formats
- AM/PM toggle for 12-hour format
- Time constraints via `minTime` and `maxTime` props
- Configurable step intervals for dropdown mode
- Full keyboard navigation support
- Accessibility support with ARIA attributes
